# Evenza

Sistema de gerenciamento de eventos em Laravel.

## Instalação

```bash
git clone https://github.com/acarolinadamore/evenza-laravel
cd evenza
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
```

## Como rodar

```bash
php artisan serve
```

Acesse: http://localhost:8000
